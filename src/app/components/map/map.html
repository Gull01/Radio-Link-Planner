📖 How to Use Radio Link Planner

🎯 Getting Started

  1. Add Points: Click anywhere on the map to place tower locations
  2. Enable Connect Mode: Click the "Enable Connect Mode" button
  3. Create Link: Click first tower (it will pulse gold), then click second tower
  4. View Analysis: Click the connection line popup and select "View Profile"
  5. Adjust Heights: Use the Elevation Profile tab to modify tower heights

🎨 Signal Strength Color Coding

Green (Strong): Signal > 70% - Excellent link quality
Yellow (Medium): Signal 40-70% - Good for basic connectivity
Red (Weak): Signal < 40% - May need amplification

💡 Pro Tips

  • Longer distances result in weaker signals
  • Clear line-of-sight improves signal strength by up to 15%
  • Increase tower heights to overcome terrain obstacles
  • Use satellite view to identify physical obstructions
