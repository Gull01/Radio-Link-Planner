# 🤖 AI Chatbot Setup Guide

## Overview
Your Radio Link Planner now includes an intelligent AI chatbot powered by **Google Gemini** (FREE tier). The chatbot provides context-aware assistance for RF planning, device recommendations, and technical guidance.

## ✨ Features

### Context-Aware Intelligence
- **Knows your project**: Reads your current points, connections, and signal data
- **Personalized recommendations**: Analyzes YOUR specific links and provides tailored advice
- **Real-time**: Updates context as you add/modify points and connections

### Capabilities
- 📊 **Link Analysis**: "Why is my signal weak between Tower A and Tower B?"
- 🗼 **Tower Height Recommendations**: "What height do I need for 10km link?"
- 📡 **Device Selection**: "What device should I use for this connection?"
- 📚 **Technical Education**: "Explain Fresnel zone clearance"
- 🔧 **Troubleshooting**: "My LOS is blocked, what should I do?"
- 🧮 **Calculations**: Quick RF calculations and estimations

## 🚀 Getting Started

### Step 1: Get Your FREE API Key

1. Visit [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Sign in with your Google account
3. Click **"Create API Key"**
4. Copy your API key (keep it safe!)

**Note**: The free tier includes:
- ✅ 15 requests per minute
- ✅ 1,500 requests per day
- ✅ NO credit card required!

### Step 2: Configure the Chatbot

1. **Start your app**: The chatbot button appears in the bottom-right corner
2. **Click the chat button**: "💬 Ask AI"
3. **Click the settings icon** (⚙️) in the chat header
4. **Paste your API key** in the input field
5. **Click "Save Key"**

That's it! You're ready to chat! 🎉

## 💬 How to Use

### Sample Questions

**Link Analysis:**
```
- Why is my signal strength only 45%?
- Can you analyze the connection between Tower A and Tower B?
- Is my link reliable for data transmission?
```

**Device Recommendations:**
```
- What device should I use for a 15km link?
- Recommend a WiFi radio for this connection
- Which device works best at 5 GHz for medium range?
```

**Technical Help:**
```
- Explain line of sight clearance
- What is Fresnel zone and why does it matter?
- How do I calculate tower height?
- What frequency band should I use?
```

**Troubleshooting:**
```
- My LOS is blocked, what can I do?
- How can I improve signal strength?
- What's causing my connection issues?
```

### Tips for Better Responses

1. **Be specific**: Mention point names, distances, or signal values
2. **Ask follow-ups**: The chatbot remembers conversation context
3. **Use project context**: Ask about "this connection" when one is selected
4. **Request details**: Ask "why" or "how" for deeper explanations

## ❓ Troubleshooting

### "Please set your API key first"
- You haven't entered an API key yet
- Click settings (⚙️) and add your key

### "Invalid API key"
- Double-check you copied the full key
- Make sure there are no extra spaces
- Generate a new key if needed

### "Rate limit exceeded"
- Free tier limit: 15 requests/minute
- Wait a minute and try again
- Consider spreading questions out

### Not getting context-aware responses
- Make sure you have points/connections added
- The chatbot updates context every 2 seconds
- Try rephrasing to mention specific elements
